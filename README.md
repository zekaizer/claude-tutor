# Claude Tutor

AI-powered tutoring system for elementary school students (grades 1-3), built with Claude Code CLI.

## Overview

Claude Tutor provides a friendly, interactive learning experience for young students. It uses Claude as the backend tutor, with a kid-friendly web interface that parents can easily set up at home.

## Features

- **Subject Support**: Math, Korean, Science, Social Studies
- **Child-Friendly**: Age-appropriate language and encouragement
- **Guided Learning**: Hints instead of direct answers
- **Safety First**: All Claude tools disabled, topic guardrails enabled
- **Conversation History**: Saved as markdown files for parent review

## Tech Stack

- **Backend**: Node.js + TypeScript
- **Frontend**: HTML/CSS/JS (React optional)
- **AI**: Claude Code CLI (`claude -p`)
- **Deployment**: Docker + Docker Compose

## Getting Started

```bash
npm install
npm run dev
```
